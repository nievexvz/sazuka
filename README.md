# Multi-Provider CDN API

API untuk mengupload file ke 15+ penyedia CDN berbeda dengan satu endpoint.

## Fitur

- 15+ penyedia CDN terintegrasi
- Semua file dilayani melalui domain Anda sendiri
- Antarmuka web yang user-friendly
- Drag & drop upload
- API RESTful
- Respons format JSON yang konsisten
- CORS enabled

## Instalasi

1. Clone repository:
```bash
git clone <repository-url>
cd cdn-api
```
